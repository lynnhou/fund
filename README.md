# fund
a fund display panel

## 一键安装

### 方法一：使用 curl（推荐）
```bash
curl -sSL https://raw.githubusercontent.com/lynnhou/fund/main/setup.sh | sudo bash
```

### 方法二：手动克隆安装
```bash
cd ~ && git clone https://github.com/lynnhou/fund.git && cd fund && sudo bash install.sh
```

### 方法三：使用本地 setup.sh
```bash
sudo bash setup.sh
```
